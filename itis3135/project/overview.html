Design Document

Project Overview

This application will display information that will help artists find which materials and tools to use when creating digital art.

The intended users of the websites will be art students or people interested in digital art, and do not know where to start.

The content of the site will mainly be a brief introduction to different digital art tools and information about these tools.

Site Map

Page Design

Home Page: Main page where users will go to. Contains information about the website, and will be easy to navigate.

About Page: Page describing contents of the site, and the makers of the site. Also contains information about how the page was made, and what the website is used for.

Contact Page: Page will contain contact information, a form to input information and a button to submit information.

Software Page: Page that contains information about various software used in digital art. Will contain javascript to show hover effects and other added functionality.

Drawing Tools Page: Page will contain information about various tools used in the digital art world. Will contain javascript and other functionality as well.

Types Of Art Page: Page will have different types of art and other information about said art. Will also include javascript functionality.

Dynamic Functionality

The website will use forms, interactive images, photo galleries, and much more with javascript.
