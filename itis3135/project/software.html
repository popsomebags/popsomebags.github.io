Software





Photoshop


Photoshop is a tool used to edit images and create effects on them. It is mostly used for overall graphic design involving images.










Illustrator


Illustrator is a tool used to create logos or images that are known as "vector images." Vector images do not lose quality when resized, and is used to create high quality scalable graphics.










Gimp


Gimp is a free alternative to photoshop and illustrator. It provides the same tools, but some features are limited and the user interface is harder to use compared to photoshop and illustrator.
